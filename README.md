# Review Count Management System (RCMS)

> **Internship Capstone Project** | Developed during internship at **DIR-ACE Technology Ltd (DAT)**

## 📌 Project Overview
The **Review Count Management System (RCMS)** is a web application designed to streamline and track project review metrics, quality assurance counts, and reporting workflows. Built with a robust **Spring Boot** backend and modern data access patterns, RCMS helps teams monitor project review progress, maintain quality standards, and generate structured analytics.

---

## 🛠️ Tech Stack & Tools

* **Backend Framework:** Java / Spring Boot
* **Data Access:** Spring Data JPA / Hibernate
* **Database:** MySQL
* **Build Tool:** Maven
* **Version Control:** Git & GitHub

---

## ✨ Key Features

* **Review Tracking:** Record, track, and manage review metrics across various project levels and tiers.
* **Project Management:** Associate review entries directly with specific project structures.
* **RESTful APIs:** Clean REST endpoints for smooth integration with frontend interfaces or third-party tools.
* **Data Persistence:** Automated schema generation and relational mapping via Spring Data JPA.
